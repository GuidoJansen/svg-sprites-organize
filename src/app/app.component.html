One new way to organize your SVG images in a sprite

See the description of this new way to organize SVG images in a sprite on Medium

Use the SVG sprites with the <use> element

We use the SVG sprites here by means of this code:

<!-- HTML -->
<svg viewBox="0 0 100 100">
  <use href="path/to/sprite.svg#anchor">
</svg>

where anchor is the id of the <g> element in the SVG sprite file.

Use SVG sprite as background-image

This one is a bit of a cheater, because we don't use background-image here, but actually we are using the sprites as a mask.

/* CSS */
.my-class {
  --webkit-mask-image: url(path/to/sprite.svg#anchor);
  mask-image: url(path/to/sprite.svg#anchor);
  background-color: gold;
}

Now we're able to use gradient fillings as well
